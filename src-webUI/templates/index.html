pyFluidBearing

Fluid Film Bearing Analysis Library

An open-source Python library for computing pressure distributions, stiffness coefficients, and damping characteristics of fluid film bearings.

Damping & Stiffness Pressure Fields
$ pip install pyfluidbearing

Bearing Analysis

Compute load capacity, stiffness, damping, and pressure distributions for fluid film bearings.

Multiple Methods

Supports Friswell and Al-Bender methods for computing K and C coefficients with different approaches.

Interactive Visualization

Generate publication-quality plots and 2D pressure field visualizations directly in your browser.

About the Project

PyFluidBearing is developed to provide easy and reproducible tools for evaluating fluid film bearing performance. The library enables comparison of analytical, numerical, and data-driven approaches for modeling and analyzing bearing behavior.

Key Objectives

  • Reproducible bearing performance evaluation
  • Multiple analytical and numerical methods
  • Open-source and extensible architecture
